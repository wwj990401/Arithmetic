# 四则运算说明文档

## 1、add

### 描述

`int add（int added，int add）`把`added`（被加数）加上`add`（加数）。

### 声明

下面是`add（）`函数的声明。

```c
int add（int added，int add）
```

### 参数

- `added` -- 被加数（int型）
- `add` -- 加数（int型）

### 返回值

该函数返回一个`int`型的值。

注意：

若返回值数据超出int值的范围，而提醒并返回0；

### 实例

下面的实例演示了`add（）`函数的用法。

```c
#include<stdio.h>

#include"../A/add.h"
#pragma comment(lib,"A.lib")

int main()
{
	int a, b;
	printf("请输入两个数：\n");
	scanf("%d%d", &a, &b);
	printf("add(%d+%d)=%d\n", a, b, add(a, b));
	return 0;
}
```

让我们编译并运行上面的程序，这将产生以下结果：

```c
请输入两个数：
1 2
add(1+2)=3
```

## 3、mul

### 描述

`int mul（int multiplied，int multiplier）`把`multiplied`（被乘数）加上`multiplier`（乘数）。

### 声明

下面是`mul（）`函数的声明。

```c
int mul（int multiplied，int multiplier）
```

### 参数

- `multiplied` -- 被乘数（int型）
- `multiplier` -- 乘数（int型）

### 返回值

该函数返回一个`int`型的值。

注意：

若返回值数据超出int值的范围，而提醒并返回0；

### 实例

下面的实例演示了`mul（）`函数的用法。

```c
#include<stdio.h>

#include <Windows.h>

typedef int(*PMUL)(int a, int b);

int main()
{
	int a, b;
	printf("请输入两个数：\n");
	scanf("%d%d", &a, &b);
	HMODULE hDLL = LoadLibrary(L"C.dll");
	if (hDLL == NULL)
	{
		printf("加载DLL文件失败。\n");
		return 0;
	}
	PMUL mul = (PMUL)GetProcAddress(hDLL, "mul");
	printf("mul(%d*%d)=%d\n", a, b, mul(2, 1));
	FreeLibrary(hDLL);
	
	return 0;
}
```

让我们编译并运行上面的程序，这将产生以下结果：

```c
请输入两个数：
1 2
mul(1*2)=2
```

## 4、div1

### 描述

`div_t div1（int numer，int denom）`把`numer`（分子）除以`numer`（分母）。

### 声明

下面是`div（）`函数的声明。

```c
div_t div1（int numer，int denom）
```

### 参数

- `numer` -- 分子（int型）
- `denom` -- 分母（int型）

### 返回值

该函数返回定义在`<cstdlib>`中的结构中的值,该结构有两个成员,如`div_t:int quot;int rem;` 。

注意：

- 若输入的分母为零，则提醒并立即返回0；

- 若返回值数据超出int值的范围，而提醒并返回0；

### 实例

下面的实例演示了`div1（）`函数的用法。

```c
#include<stdio.h>
#include"div.h"

int main()
{
	int a, b;
	div_t output;
	printf("请输入两个数：\n");
	scanf("%d%d", &a, &b);
	output = div1(a, b);
	printf("div(%d/%d)=%d余%d\n", a,b,output.quot,output.rem);
	return 0;
}
```

让我们编译并运行上面的程序，这将产生以下结果：

```c
请输入两个数：
1 2
div1(1/2)=0余1
```
